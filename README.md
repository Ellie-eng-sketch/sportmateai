# Overview

SportAI helps athletes, students, and sports enthusiasts find teammates, organize games, discover nearby sports facilities, and improve participation through artificial intelligence.

- Team Formation
- Event Creation
- Nearby Match Finder
- AI Matchmaking
- Performance Tracking
- Group Management
- Notifications
- AI Assistant
